What is the command line?

  • The command line, or command line interface (CLI) is a thing. All operating systems have some sort of command line. On MacOS and Linux the CLI app is called the Terminal. WindowsOS has three CLI apps - the Command Prompt, Windows PowerShell, and Windows PowerShell ISE (Integrated Scripting Environment).
  • The main difference is the command prompt is the oldest of the three and used for more simple tasks whereas the powershells are used for more complicated tasks.

IP addresses and MAC addresses

There are two main types of IP addresses:

  • Private IP Addresses:
  • Not visible to anyone on the Internet
  • Used to identify a machine on a local device
  • assigned by router
  • Addresses can be reused on different networks
  • free
  • Can be found from the command line when connected to a router

Finding private IP address from command line
MacOS WindowsOS Purpose
ifconfig (private IP is IPv4) ipconfig (private IP is IPv4) find private IP address
  • Public IP Addresses:
    • Visible to anyone on the Internet
    • Used to identify your device when you are connected to the Internet
    • assigned by your ISP
    • Addresses are unique and cannot be reused
    • not free, have to pay an ISP for them
    • Can be found when connected to the internet
    • different devices on same network have same public IP

    You can't find public IP address without being connected to the internet. This is because public IP addresses are assigned by ISP. However, once connected to the internet you can find your public IP from the command line.
