What is the command line?

The command line, or command line interface (CLI) is a thing. All operating systems have some sort of command line. On MacOS and Linux the CLI app is called the Terminal. WindowsOS has two CLI apps - the Command Prompt and Windows PowerShell.

Basic Commands

Terminal (MacOS) Command Prompt (WindowsOS) Purpose
cd cd change directory
pwd pwd print working directory
ls dir list contents of working directory
cd dir1/dir2 cd dir1\dir2 change more than one directory level
cd .. cd .. step up one directory
cd cd\ go back to root directory
mkdir [name] mkdir [name] create directory
rmdir [name] rmdir [name] remove empty directory
rm -rf [name] rm -R [name] delete directory and its contents
touch [name.type] NO GOOD WAY create empty file
touch [name.type] echo [text] > file.type create non-empty file
rm [name.type] del [name.type] delete file
