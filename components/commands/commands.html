What is the command line?

The command line, or command line interface (CLI) is a thing. All operating systems have some sort of command line. On MacOS and Linux the CLI app is called the Terminal. WindowsOS has two CLI apps - the Command Prompt and Windows PowerShell. The command prompt is used for more simple tasks whereas the powershell is used for more complicated tasks.

SSH (Secure Shell)

The first step to logging in with SSH is knowing the public IP address of the target device. There are two main types of IP addresses:

  • Public IP Addresses:
    • Visible to anyone on the Internet
    • Used to identify your device when you are connected to the Internet
    • Can be dynamic or static
  • Private IP Addresses:
    • Not visible to anyone on the Internet
    • Used to identify a machine on a local device
    • Can be dynamic or static

MacOS WindowsOS Purpose
curl ifconfig.me sdg find public IP address
