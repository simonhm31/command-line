Tables:

The <table></table> tags are used for displaying tabular data

Tags used inside of <table></table> are:

  • <tr><tr>: "table row"
  • <td><td>: "table data cell"
  • <thead><thead>: "table header"
  • <tbody><tbody>: "table body"
  • <th><th>: "table header element"
